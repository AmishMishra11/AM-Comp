Buttons

Buttons are the clickable things that perform a specific task or take you to a perticular webpage.

Primary Buttons

There are two types of buttons available, use class button on the parent container and use class button-primary for primary button and button-secondary for secondary buttons

Links

There are two types of links available, use class link on the parent container and use class primary-link for primary links and secondary-link for secondary links.

Primary Secondary

Icon

Sometimes we need icons as linking different pages of our websites. For designing icons as links use class icon on the parent container.

Floating Action Button

If you want to display a floating button on your web page use class fab and inside use a button and it will appear as a floatin button.
